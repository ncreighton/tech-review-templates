# Tech Review Templates

Site-specific review page templates and component library for tech review sites.

## Sites Covered
- **PulseGearReviews.com** - Fitness tech (Energy Red theme)
- **WearableGearReviews.com** - Wearables (Health Teal theme)
- **SmartHomeGearReviews.com** - Smart home (Trust Blue theme)

## Components
- Quick verdict boxes
- Fitness metrics cards
- Health sensor accuracy displays
- Battery endurance charts
- Ecosystem compatibility matrices
- Setup difficulty ratings
- Specs tables
- Final verdict boxes
